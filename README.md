# docker-wordpress

## これは何?

WordPressのテーマ, プラグインを開発するために使えるDockerコンテナ.

## 使い方

docker-compose.ymlのあるディレクトリに移動して以下を実行.

動かす時: `docker-compose up -d`

止める時: `docker-compose down`
